Axon WebHID PoC

Minimal browser probe for the Axon V1.3 HID adapter. This page requests the device, opens it over WebHID, sends the identify command, and can read the full 95-byte config block.

  • Use a Chromium-based browser on https:// or http://localhost.
  • Make sure the adapter is owned by the host OS, not a VM such as Parallels.
  • The PoC uses report id 0x04 with a 63-byte payload.
